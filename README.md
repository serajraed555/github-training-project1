# Login Interface Project (Git & GitHub Training)

## Project Description
Simple Login Interface project for Git & GitHub training.

## Technologies Used
- Java
- XML
- Server (Backend)

## Features
- Login Screen (UI)
- Input Validation
- Basic Authentication Flow
- Responsive Layout (Android UI)

## How to Run
1. Open the project in Android Studio.
2. Sync Gradle files.
3. Run the application on an emulator or real device.
